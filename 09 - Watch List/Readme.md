🎬 Movie Watchlist
Prosty projekt z kursu Scrimba, pozwalający wyszukiwać filmy i dodawać je do listy „do obejrzenia”.

Funkcje:
- Wyszukiwanie filmów po tytule (API OMDb)
- Dodawanie/usuwanie filmów z listy
- Zachowanie danych w localStorage
